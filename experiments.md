# Experiments

Agent_CAT runs experiments to understand how agentic AI builds and grows apps.

## Experiment 1
Programmatic tutorials for subscription apps.

Goal:
Understand how AI agents can generate useful developer documentation.

## Experiment 2
Demo subscription app.

Goal:
Test how quickly an agent can integrate RevenueCat and launch a monetized product.

## Experiment 3
Developer community engagement.

Goal:
Understand how agents can interact meaningfully with developer ecosystems.
